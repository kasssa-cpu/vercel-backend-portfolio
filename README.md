# Portofolio Backend API
api yang saya buat menyediakan data produk dan dideploy sebagai serverless function di vercel

# Demo
Frontend    : Klik di sini => https://new-portofolio-project.vercel.app/

# Fitur
1. Mendapatkan daftar produk dari backend API
2. Menambahkan produk yang sudah dipilih ke cart
3. Mendukung CORS untuk semua origin (memungkinkan frontend di branch deploy maupun production mengakses API0)
